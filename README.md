# MyReads Project

This project provides a handy user interface to organize the user books in three categories:

- Currently Reading
- Want to Read
- Read

The user can freely move a book from one shelf to another.

There user also can add new books by navigating to the search page from the bottom right button


##Running steps

- steps to install the project dependencies:

You will need to run the following commands:

npm add react-scripts
npm install --save react-router-dom

-steps to run the projects:

npm start
